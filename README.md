# Site Vitrine pour Kristyle Lenoir - Créatrice de Mode

Site web élégant et haut de gamme pour Kristyle Lenoir, créatrice de mode, dessinatrice, couturière, styliste et modéliste.

## 🎨 Identité Visuelle

- **Palette** : Noir profond (#0a0a0a), Violet doux et royal (#6B21A8 / #9333EA), Or chaud (#C9A84C), Blanc cassé (#F5F0EB)
- **Typographie** : Playfair Display (titres) + Raleway (corps)
- **Style** : Haute couture, minimaliste luxe, aéré

## 📋 Structure du Site

- **Accueil** : Hero avec animation de particules
- **À Propos** : Présentation de la créatrice et timeline
- **Créations** : Galerie filtrée par catégorie
- **Collections** : Lookbook des collections saisonnières
- **Services** : Sur-mesure, consultation, ateliers
- **Actualités** : Blog et revue de presse
- **Contact** : Formulaire et rendez-vous

## 🚀 Installation

1. Clonez ce repository
2. Ouvrez `index.html` dans votre navigateur
3. Pour les modifications, éditez les fichiers :
   - `css/style.css` pour le style
   - `js/main.js` pour les interactions

## 💻 Technologies Utilisées

- HTML5
- CSS3 (variables CSS, Flexbox, Grid)
- JavaScript Vanilla
- AOS (Animate On Scroll)
- Font Awesome Icons

## 📱 Responsive Design

Le site est entièrement responsive et s'adapte à tous les écrans :
- Desktop (1200px+)
- Tablet (768px - 1199px)
- Mobile (moins de 768px)

## 🌐 Hébergement

Le site peut être hébergé sur :
